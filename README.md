This is a pet-project web app that was created with ejs,bootstrap,express and local postgres db.
The app allows users to see profile, upload, search and download several kinds of music-related products.
If you want to try to run it yourself, you can follow these steps:
Make sure you have npm and node.js installed
Clone the repository
In you database manager (e.g. pgAdmin 4), use the museground_dump.sql file to "restore" a database
In your IDE:
run npm install
run npm run dev
open localhost:3000 in you browser

In the app, you must register, and then login.
When you login, you would be able to see your profile, 
upload samples and other products, search and filter through various products and download them.
